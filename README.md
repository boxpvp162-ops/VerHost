
# VerHost – Minecraft Hosting Platform

## Technologie
- Frontend: HTML, CSS, JS
- Backend: Node.js (API pod Vercel)
- Baza danych: Supabase

## Funkcje
- Rejestracja / Logowanie
- Panel klienta
- Panel admina
- Oferty hostingowe
- Generator serwerów (realny – do podpięcia np. Pterodactyl API)

## Instalacja
1. Stwórz projekt w Supabase
2. Uzupełnij dane w `api/config.js`
3. Wrzuć projekt na GitHub
4. Zaimportuj repozytorium do Vercel
